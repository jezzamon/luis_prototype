    Installing Java

    Java must be installed on this client in order to view the management dashboards, submit VTT, or launch installer. If Java is already installed, it will be used automatically.
    If Java is not installed, an attempt will be made to download and install Java automatically. You can install manually by downloading Java software from the Java website.
